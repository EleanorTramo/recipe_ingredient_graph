How It Works

This tool uses a knowledge graph — a network of nodes (ingredients, cuisines, and flavor profiles) connected by weighted edges. Two ingredients are good substitutes if they share many of the same connections with high-strength weights.

The substitution score averages edge weights along each shared path between two ingredients. Scores are ranked and labeled: High match ≥ 70%, Good match ≥ 40%, Possible below that.
